    Τα αρχεία που δημιούργησα βρίσκονται στον φάκελο myfiles, το πρόγραμμα τρέχει σύμφωνα με τις απαιτήσεις τις εργασίας,
η makefile πέρα από την make έχει και τις: make clean (καθαρίζει τα .class) αρχεία, make extra (τεστάρει τα extra 
αρχεία*), make errors (τεστάρει τα errors αρχεία), make examples (τεστάρει τα παραδείγματα*).
    Ο πρώτος visitor τοποθετεί τα πάντα στους κατάλληλους πίνακες και ελέγχει για διπλοτυπίες ενώ ο δεύτερος ελέγχει για
τα λοιπά λάθη που δεν μπορούν να ελεγχούν στον πρώτο.
    Χρησιμοποιησα LinkedHashMap για να αποθηκεύσω τις συναρτήσεις, τις μεταβλητές και τις τάξεις ενώ χρησιμοποίησα
πίνακα για τα arguments των συναρτήσεων.

$make
$java Main [ονόματα αρχείων .java]
